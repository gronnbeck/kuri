## Walk-and-talk audio module

Combine conversation exercise audio clips into a single long audio file for walk-and-talk practice.

- Select which exercises to include (all or a subset)
- Concatenate request + response audio with a configurable pause between them (for thinking time)
- Add a configurable gap between each exercise
- Use a Ruby gem or ffmpeg to stitch the audio
- Output is a downloadable MP3
